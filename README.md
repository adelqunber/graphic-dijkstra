# Operating Systems Project - Graph Movement Simulation

## Description

This project simulates movement on a directed weighted graph.

The project is built in C on Linux.

## Milestone 1

Milestone 1 implements:

- Directed weighted graph representation
- Reading the graph from a text file
- Running Dijkstra shortest path algorithm
- Printing the full shortest path
- Printing the total path weight
- Handling invalid negative input
- Handling the case where no path exists
- Handling the case where source equals destination

### Compile Milestone 1

```bash
make milestone1
